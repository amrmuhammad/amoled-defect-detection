### Paper Abstract

AMOLED displays dominate consumer electronics, but manufacturing defects 
significantly reduce yield. We present AMOLED-DefectNet, a transfer learning 
system using MobileNetV2 that achieves **100% accuracy** across five defect 
categories (dead pixels, stuck pixels, Mura, scratches, dust) with <0.1s 
inference on CPU.
